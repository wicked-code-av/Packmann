# Konfiguration für das Spiel

# Größe der Kacheln, aus denen sich das Spielfeld zusammensetzt, in Pixeln
kacheln 16 px (Default 16 px)

# Geschwindigkeit Packmann
geschwindigkeit_packmann 4 (Default: 1)

# Geschwindigkeit Geister
geschwindigkeit_geister 0 (Default: 1)

# Für die Geschwindigkeiten gilt: Sie müssen ein Teiler von kacheln sein, sonst kommt es zu glitching.
# Das Spiel überschreibt ungültige Werte mit Defaults.
